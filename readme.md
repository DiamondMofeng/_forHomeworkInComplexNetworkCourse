analyze.py可以导入网络的节点、边数据,对其统计指标以及鲁棒性进行分析

# train

train.py可以输入城市列表，自动匹配含有此城市名的车站，并从12306爬取数据，输出车站对之间对应车次信息

process_train.py可以输入train.py的结果与 城市列表 文件，输出城市间的有向边数据。权重为当日此城市对之间的车次数量。

getDict.py可以获取12306的车站字典

# bus
